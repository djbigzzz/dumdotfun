## Option 3: Run Locally (10 minutes)

### Prerequisites
- Node.js 18+
- PostgreSQL
- Helius API key ([free tier](https://helius.dev))

### Quick Setup
```bash
# 1. Clone
git clone <repository-url>
cd job

# 2. Install
npm install

# 3. Create .env
cat > .env << EOF
DATABASE_URL=postgresql://user:password@localhost:5432/dumfun
HELIUS_API_KEY=your-key-here
SOLANA_NETWORK=devnet
EOF

# 4. Database
npm run db:push

# 5. Run
npm run dev
```

Open http://localhost:5000 and connect your Phantom wallet (set to Devnet).
